Java Function接口

一、@FunctionalInterface注解

@FunctionalInterface注解是JDK1.8中许多老的接口都增加上了这个注解，当接口标注了该注解，这个接口会被称为函数式接口。

函数是接口只能有一个自定义的方法，从Object继承来的接口可以使用但是不能够覆盖，否则不算是函数式接口。当我们使用@FunctionalInterface注解但接口不是函数式接口的格式则会抛出异常，如果函数式接口格式的接口没有标注该注解不会对其函数式使用有影响。

还有就是在JDK8接口中，静态方法和默认方法都不算是抽象方法例如Function接口中就有静态方法和默认方法，关于这两个后面再说
